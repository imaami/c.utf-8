Check if an integer value is negative without getting warning spam if the type of the value is unsigned.

If x is an unsigned integer type the macro expands to what should be a compile-time constant expression 0, otherwise to x < 0. In the latter case x is assumed to be a signed integer.

Note
The _Generic expression used to implement this macro only has explicit cases for the old school C unsigned integer types and a default case for everything else.
